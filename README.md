# CitizenAssistApp

CitizenAssistApp is a Government Grievance Redressal System designed to allow citizens to register complaints online and offline.

## Features

- Voice Complaint Registration
- Multi-language Support
- Complaint Tracking
- Admin Dashboard
- Offline Grievance Support
- Secure Backend with Flask
- SQLite Database Integration

## Project Structure

backend/        -> Flask API  
frontend/web/   -> HTML, CSS, JS  
database/       -> SQLite DB  
localization/   -> Language support  

## Technologies Used

- Python (Flask)
- HTML5
- CSS3
- JavaScript
- SQLite
